ОПИСАНИЕ:
SIK2D - это легковесный 2D фреймворк для Lua, работающий без
использования внешних библиотек. Он использует виртуальный холст
из символов ASCII и выводит результат в консоль.

УСТАНОВКА:
1. Скопируйте файл SIK2D.lua в папку с вашим проектом
2. Подключите модуль: local SIK2D = require("SIK2D")

ИСПОЛЬЗОВАНИЕ:

local SIK2D = require("SIK2D")
SIK2D.init(80, 24)                    -- Инициализация холста
SIK2D.setPixel(10, 5, '@')             -- Рисование точки
SIK2D.drawLine(1, 1, 10, 10, '#')      -- Линия
SIK2D.drawRect(5, 5, 20, 10, '*', true)-- Прямоугольник (с заливкой)
SIK2D.drawText(2, 2, "Hello!")         -- Текст
SIK2D.render()                          -- Вывод на экран
SIK2D.saveToFile("output.txt")          -- Сохранение в файл

ФУНКЦИИ:
- init(width, height)              - инициализация
- clear()                          - очистка холста
- setPixel(x, y, char)             - установка символа
- getPixel(x, y)                    - получение символа
- drawLine(x1, y1, x2, y2, char)   - линия
- drawRect(x, y, w, h, char, fill) - прямоугольник
- drawText(x, y, text)             - текст
- drawBorder(char)                  - рамка вокруг холста
- render()                          - вывод в консоль
- saveToFile(filename)              - сохранение
- loadFromFile(filename)            - загрузка
- getSize()                          - размеры холста
- setClearChar(char)                 - символ для очистки

ПРИМЕЧАНИЯ:
- Все координаты начинаются с 1 (левый верхний угол)
- Символы обрезаются до первого символа
- Файлы export/ и visual/ содержат заглушки для будущих модулей
